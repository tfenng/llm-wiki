# llm-wiki Alfred Workflow

Search your llm-wiki knowledge base from Alfred.

## Features

- Type `wiki <query>` in Alfred to search across all wiki pages
- Uses `search-index.json` (generated by `llmwiki build`) when available
- Falls back to scanning wiki/ folder titles
- Select a result to open it in your browser at localhost:8765

## Prerequisites

- [Alfred](https://www.alfredapp.com/) with Powerpack
- Python 3.9+
- An llm-wiki project on disk
- Run `python3 -m llmwiki build` at least once to generate the search index

## Install

1. Double-click `info.plist` to import into Alfred, or:
   - Open Alfred Preferences > Workflows
   - Drag the `alfred/` folder into the workflow list
2. Set the **LLMWIKI_DIR** workflow variable to point at your llm-wiki project root
3. Type `wiki` in Alfred to start searching

## How it works

The workflow uses an Alfred Script Filter that runs `search.py` with your query. The script loads `site/search-index.json` and performs case-insensitive substring matching across page titles, paths, and body text. Results are returned as Alfred JSON feedback items that open in the browser when selected.

## Files

| File | Purpose |
|---|---|
| `info.plist` | Alfred workflow definition (Script Filter -> Open URL) |
| `search.py` | Python search script called by the Script Filter |
