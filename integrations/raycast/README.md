# llm-wiki Raycast Extension

Search and browse your llm-wiki knowledge base from Raycast.

## Features

- **Search Wiki** -- full-text search across all wiki pages
  - Uses `search-index.json` (generated by `llmwiki build`) when available
  - Falls back to filesystem grep over `wiki/` markdown files
- **Open Wiki** -- opens the local site in your browser

## Prerequisites

- [Raycast](https://raycast.com) installed
- An llm-wiki project on disk
- Run `python3 -m llmwiki build` at least once to generate the search index

## Install (development)

```bash
cd integrations/raycast
npm install
npm run dev
```

This opens the extension in Raycast development mode. Configure the **Wiki Directory** preference to point at your llm-wiki project root.

## Configuration

| Preference | Description |
|---|---|
| Wiki Directory | Path to the llm-wiki project root (required) |
| Server Port | Port for the local server (default: 8765) |
