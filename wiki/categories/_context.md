# wiki/categories/

Auto-generated tag-based index pages. Each file is either a Dataview
query page (for Obsidian) or a static markdown list (for the GitHub
Pages site). Generated by `llmwiki.categories`.

**When to descend here:** The user asks about a specific topic tag
(e.g. "what's tagged as 'flutter'?") and wants a roll-up view.

**When to skip:** Direct entity/concept/source lookups route through
their respective folders. Categories are a secondary index.
